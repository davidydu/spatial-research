## 1. Purpose

The **Forge + shared runtime** bundle supplies the Scala macro machinery, the cross-cutting utilities, and the numeric emulation layer that all higher layers of Spatial depend on. Three sibling packages:

- **`forge`** (`forge/src/forge/`) — macro annotations (`@op`, `@ref`, `@api`, `@ctx`, `@stateful`, `@rewrite`, `@globalRewrite`, `@flow`, `@virtualize`, `@curriedUpdate`, `@data`, `@mod`, `@rig`, `AppTag`) and the minimal runtime hooks they emit against (`SrcCtx`, `AppState`, `VarLike`, `Ptr`, `Tri`, `EmbeddedControls`). These macros run at compile time on `argon` / `spatial` sources; they rewrite class/def trees to inject IR-construction calls and implicit parameters (`SrcCtx`, `State`).
- **`utils`** (`utils/src/utils/`) — shared helpers: `Instrument`/`Instrumented` (phase-level timing), `Tree.LCA` (ancestor/LCA over arbitrary parent functions), `MacroUtils` (Scala `reflect` wrapper used by every `forge` annotation), `implicits.collections`/`terminal`/`Readable`, `math` package (log2, gcd, Mersenne helpers, banking math `computeP`), subprocess wrappers, `Result`/`Testbench` types, `Args`, `Overloads`, `recursive`, `stats`, CSV/path `io.files`.
- **`emul`** (`emul/src/emul/`) — bit-accurate runtime for the Scala simulator generated by `spatial.codegen.scalagen.*`: `FixedPoint`/`FixFormat`, `FloatPoint`/`FltFormat`, `Bool`, `Number`, `Counter`, `Memory`/`BankedMemory`/`ShiftableMemory`/`LineBuffer`, `Stream*`, `DRAMTracker`, `OOB`, `StatTracker`, `ResidualGenerator`.

Pipeline-wise, these are **foundation** layers, not pipeline phases: the `forge` macros act during Scala compilation of the Spatial compiler itself (and of the user app once virtualized), `utils` is pure support, and `emul` is linked by generated Scala simulators at runtime — it never participates in compilation.

## 3. Key types / traits / objects

- **`forge.SrcCtx`** (`forge/src/forge/SrcCtx.scala:5-19`). Value class threaded as implicit through essentially every user-facing and IR-construction method. - **`utils.Instrument` / `utils.Instrumented`** (`utils/src/utils/Instrument.scala`, `utils/src/utils/Instrumented.scala`). Hierarchical timer. `Instrument.apply(name){blk}` builds dotted scope keys; `fix` propagates subtotals upward. `Instrumented.set` is the global registry written to by every timed class via `@instrument`.

## 6. Key algorithms

- **Scala-virtualization tree rewrite** — `forge/src/forge/tags/Virtualizer.scala:76-286`. Pattern matches on `Template`/`Block`/`Function`/`If`/`Return`/`LabelDef` (for `while`/`do-while`) and all `Any`/`AnyRef` infix methods. Lifts mutable `var` to `__newVar` and registers `__valName`.
- **Source-context macro injection** — `forge/src/forge/tags/SrcCtxMacro.scala:8-20` reads `c.enclosingPosition` and builds `forge.SrcCtx(path,filename,line,column,lineContent,Seq.empty)`.
- **Implicit-parameter injection across def lists** — `utils/src/utils/tags/MacroUtils.scala:117-126` (`injectImplicit`) merges into the existing implicit parameter list only when no `equalsStructure`-matching type is already there.
- **Ref class boilerplate emission** — `forge/src/forge/tags/ref.scala:37-48` builds `cargs`, `fresh`, `__typePrefix`, `__typeArgs`, `__typeParams` and selects one of four `tp` companion shapes based on (vparams,iparams) emptiness.
- **Op mirror/update synthesis** — `forge/src/forge/tags/op.scala:30-48` emits `override def mirror($f:Tx)` calling `$f` on each constructor arg, and `update($f:Tx)` reassigning each field in place.
- **CurriedUpdate dispatcher** — `forge/src/forge/tags/curriedUpdate.scala:49-58` builds a whitebox macro dispatcher so `update(...)` with variadic arguments routes to the fixed 3-list form `update$r(inds...)(value)(ctx,state)`.
- **Rewrite / flow registration** — `forge/src/forge/tags/rewrite.scala:48-62`, `globalRewrite.scala:48-64`, `flow.scala:41-53`. Materializes a `PartialFunction[(Op[_],SrcCtx,State),Option[Sym[_]]]` (or `(Sym,Op,...)=>Unit` for flow) and registers it in the compiler's rule tables.
- **Reduce-tree + combination-tree** — `utils/src/utils/math/ReduceTree.scala:5-12` (balanced binary reduce) and `CombinationTree.combine` (`ReduceTree.scala:30-40`).
- **Banking P search** — `utils/src/utils/math/package.scala:134-209` (`computeP`). Core recipe: `P_raw_i = N·B / gcd(N·B, α_i)`; expand with divisors and `stagedDim`; filter by volume and census constraint `spansAllBanks`.
- **Crandall division by Mersenne** — `utils/src/utils/math/package.scala:73-101` (`modifiedCrandallSW`). Iterative `q/r` reduction given `t = 2^k − c`.
- **LCA over generic trees** — `utils/src/utils/Tree.scala:24-44`. `zip(reverse).find(a==b)` over `ancestors`.
- **Instrument hierarchical aggregation** — `utils/src/utils/Instrument.scala:39-53` (`fix`). Builds subtotals from leaf paths upward via prefix matching.
- **Float clamp / mantissa-exponent packing** — `emul/src/emul/FloatPoint.scala:318-398` (`clamp`). Handles subnormals, denormal flushing, biased exponent.
- **Unbiased rounding** — `emul/src/emul/FixedPoint.scala:232-241` (`FixedPoint.unbiased`). Uses `Random.nextFloat()` + 4 extra fractional bits.
- **Circular line-buffer update** — `emul/src/emul/LineBuffer.scala:43-60`. `posMod` on row index + `wrCounter`/`lastWrRow` state to track partial rows.
- **Residual expansion** — `emul/src/emul/ResidualGenerator.scala:15-19` (`expand(max)`). Case-analysis on `M == 0` vs `A == 0` vs general.

## 7. Invariants / IR state read or written

- **`SrcCtx` chaining invariant** — `SrcCtx.+(other)` appends to `previous` without replacing the head (`forge/src/forge/SrcCtx.scala:14-18`). Transformers that want "A came from B" should `newCtx + oldCtx`, never swap.
- **`@virtualize` does not virtualize params or type aliases** — `virtualize.impl` (`forge/src/forge/tags/virtualize.scala:27-37`). Relied on by macros that wrap the result of `AppTag.impl`.
- **`@op` rewrites vparams to `var`** via `cls.asCaseClass.withVarParams` (`forge/src/forge/tags/op.scala:32`). Downstream `update($f)` depends on those fields being reassignable.
- **`@ref` assumes the class mixes in `Ref[?,A]`** — comment at `forge/src/forge/tags/ref.scala:27`. Not structurally checked.
- **`@flow` enforces signature `(Sym[_], Op[_])`** and wildcards the type args via `isWildcardType` (`forge/src/forge/tags/flow.scala:34`).
- **`@rewrite` forbids implicits, forbids type params, requires the body to be a `Match`** (`rewrite.scala:36-46`).
- **`@curriedUpdate` requires exactly 3 vparam lists** of sizes `(≥1, 1, 2)` (`curriedUpdate.scala:22-28`). The last list must carry `ctx` and `state`.
- **`Instrument.apply` de-duplicates recursive nesting**: if the scope already contains `name`, the block runs without double-counting (`Instrument.scala:19-23`). This is important for transforms that recurse into themselves.
- **`Instrumented.set`** is an `ArrayBuffer` — inheritors accumulate forever; reset is per-instance via `resetInstrument()`.
- **`emul.Bool` is three-valued**: `valid=false` produces `X` bits (`Bool.scala:17`, `Number.bitString`). FixedPoint/FloatPoint propagate `valid` through every operation (`this.valid && that.valid`).
- **`FixedPoint.clamped` wraps on overflow**, `FixedPoint.saturating` saturates, `FixedPoint.unbiased` rounds with RNG (3 distinct constructors selected by op variant `+`/`+!`/`*&`).
- **`OOB` logs are *not* append-safe across runs** — `./logs/reads.log`/`./logs/writes.log` opened unconditionally (`OOB.scala:7-8`).
- **`StatTracker.enable.top` defaults to `false`** — must `pushState(true)` before stats record (`StatTracker.scala:6`).
- **`DRAMTracker.accessMap` is a JVM-global `mutable.Map`** (`DRAMTracker.scala:6`). Simulations that share a JVM will see cross-talk.
- **`utils.recursive.collectSeq` traverses `Product`s**, so it will walk into Scala case classes by default — consumers relying on this pass IR nodes.

## 8. Notable complexities or surprises

- **`Virtualizer` has a commented-out `Match` rewrite** (`forge/src/forge/tags/Virtualizer.scala:248-279`) and an explicit "@throws is currently unsupported" pragma around `Try` (`:166-169`). Plus a "NOFIX" note on `"string" + unstaged + staged` (`:176-179`). Real issue list; flag for Phase 2.
- **`@globalRewrite` always prints `showCode` with `force = true`** (`forge/src/forge/tags/globalRewrite.scala:70`). Noise during compilation; surprisingly not gated.
- **`@mod` is essentially empty** — it keeps the class unchanged but still has `info(showCode(tree), force=true)` (`mod.scala:34`). The commented-out `injectMethod("names")` suggests this is vestigial.
- **`forge.EmbeddedControls.anyVal_+`** tries to infer widest weak type (`EmbeddedControls.scala:267-281`) and emits a compile error if neither side widens — a rarely-triggered macro hook.
- **`EmbeddedControls.DefaultAnyMethods.infix_isInstanceOf`** uses `utils.isSubtype(lhs.getClass, classTag[T].runtimeClass)`; erasure-safe but doesn't check generic type args.
- **`utils.implicits.collections.SeqHelpers.pairs`/`triplets`** assume the source collection has distinct elements (comment at `collections.scala:115-116`); duplicate elements would be pairwise-combined incorrectly.
- **`utils.math.computeP` has a long prose comment documenting corrections to Wang et al. FPGA '14** — see `math/package.scala:134-209`. Phase 2 should treat this as a "paper corrigendum" note.
- **`ResidualGenerator.expand` has a `NOTE: Not true after area modeling changes`** comment at `ResidualGenerator.scala:17` — known discrepancy with compiler-side `ResidualGenerator` metadata.
- **`emul.Counter.takeWhile` prints `continue?` to Console on every iteration** (`Counter.scala:27`). Looks like leftover debug.
- **`FloatPoint.clamp` contains numerous commented-out `println` debug lines** — not noise-free if uncommented.
- **`FixedPoint.unbiased` calls `Random.nextFloat()` on each operation** (`FixedPoint.scala:236-237`) — nondeterministic rounding in simulation unless `Random.setSeed` is explicitly set by the caller.
- **`BufferedProcess.checkErrors` raises on any stderr output** (`BackgroundProcess.scala:23-31`). Subprocesses that use stderr for progress will break.
- **Two `Ptr` classes** — `forge.Ptr[T]` (extends `VarLike`) and `emul.Ptr[T]` (simulator scalar container). They have near-identical shape but different concerns; easy to confuse.
- **`SrcCtx.empty` has `file = "?"`** — any downstream code that parses `file` for path operations must guard against the literal `"?"`.
- **`FixedPoint.toShort` shifts by `fmt.bits`, not `fmt.fbits`** (`FixedPoint.scala:81`). Looks like a bug — inconsistent with the `toByte`/`toInt`/`toLong` sibling methods that all shift by `fbits`.
- **`@rewrite` vs `@globalRewrite`**: `rewrite` dispatches to `IR.rewrites.add[T]` or `addGlobal` depending on whether the arg type starts with `Op[` (`rewrite.scala:57-62`). `globalRewrite` always uses `addGlobal` — but without the `isInstanceOf` safety check in the generated partial function (compare `rewrite.scala:50-55` with `globalRewrite.scala:49-58`). Known coupling: Phase 2 should trace rule firing order.

## 9. Open questions

- What is the precise semantics of `SrcCtx.previous`? Who appends to it, and is the order "oldest first" or "most recent first"?
- Where exactly do `@op`'s synthesized `mirror`/`update` get called from inside the transformer pipeline (Tx lookup)?
- `@mod` appears to be a stub — is there a richer historical implementation elsewhere, or is this simply a placeholder for future module typing?
- `utils.OutlierFinder` is reachable but I did not find callers via sampling — is it wired into the instrumentation pipeline, or orphaned?

- `FixedPoint.toShort`'s apparent bit-shift error (`fmt.bits` vs `fmt.fbits`) — is this intentional for signed 16-bit extraction or a real bug?
- `FloatPoint.clamp`'s handling of `y < fmt.SUB_E && x > 1.9` (`FloatPoint.scala:335-339`) uses magic number 1.9; what's the justification?
